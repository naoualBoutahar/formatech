# formatech
une application Mobile avec IONIC qui permet de réaliser des réservations pour des formations en technologies de développement informatique. Ces formations sont proposées par un centre qui liste toutes les formations disponibles, leurs dates, une description et le prix.
Le client potentiel peut consulter le détail de chaque formation et avoir la possibilité de faire une réservation. Celle-ci est sauvegardé avec son profil dans une BDD.
